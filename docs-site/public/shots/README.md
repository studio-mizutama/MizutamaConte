# ランディング用スクリーンショット置き場

ここ（`docs-site/public/shots/`）に下記のファイル名で置くと、ランディングに自動で反映されます。
**Electron 版を 16:9 ウィンドウ（1600×900）で起動して、ウィンドウを撮影**してください。
（英語UI・ライトテーマ推奨。ダークは切替デモを撮るときだけ。）

| ファイル名 | 形式 | 中身 |
|---|---|---|
| `hero.mp4` | 動画(16:9・無音ループ) | 編集タブでスクロール → プレビュータブに切替 → 再生、の画面収録。ヒーロー。 |
| `feat-psd.png` | 静止画(16:9) | PSD ネイティブ往復。カット＋PICTURE が見える状態（外部画ソフトと並べてもよい）。 |
| `feat-camera.png` | 静止画(16:9) | カメラワーク／クロップ。クロップツール or カメラワークパネルが効いている状態。 |
| `feat-stopwatch.png` | 静止画(16:9) | ストップウォッチ Rec。T ツール・行の Rec・TIME カウント中など。 |
| `feat-script.png` | 静止画(16:9) | 脚本から生成されたカット（セリフが入った盤面）。 |
| `feat-export.png` | 静止画(16:9) | 書き出し。印刷/PDF プレビュー、または書き出しダイアログ。 |

- すべて **16:9**（ウィンドウが 1600×900 なのでウィンドウ全体を撮ればOK）。
- 置けば差し替え不要で表示されます。未配置の枠はファイル名つきプレースホルダのまま。
- 実用ページ（使い方等）に図を足したくなったら、同じ `shots/` に置いて md から `![](shots/xxx.png)` で参照できます（base 自動付与）。
