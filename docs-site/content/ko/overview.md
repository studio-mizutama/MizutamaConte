# 소개

**Mizutama Conte** 는 콘티(스토리보드)를 제작하고 관리하는 데스크톱／Web 앱입니다.

가장 큰 특징은 **카메라 워크(팬·줌)가 적용된 비디오 콘티(애니매틱)를 미리 볼 수 있다**는 점입니다. 레이아웃을 나열하는 데 그치지 않고, 실제로 움직이는 영상으로 콘티를 확인할 수 있습니다. 이는 유사 도구에서 보기 드문 기능입니다.

## 주요 기능

- **해상도·화면비**：SD / HD / FHD / 2K / 4K 와 4:3 / 16:9 / 1.85:1 / 2.39:1 을 조합하여 새 프로젝트 생성
- **편집(Edit)**：행 추가, 씬 구분, CUT 내 여러 레이어(시계열 플립북), 🔗 CUT 결합·분리, ACTION / DIALOGUE / TIME 인라인 편집
- **카메라 워크**：CUT 마다 팬·줌 설정. 슬라이더와 수치 입력으로 조정
- **트랜지션**：페이드인／페이드아웃, 인접한 CUT 사이에서 공유되는 크로스
- **캔버스 크롭·리사이즈**：리사이즈 시 방향에 맞는 기본 카메라 워크를 자동 생성
- **외부 드로잉 앱 연동**：PICTURE 를 더블클릭하여 CLIP STUDIO PAINT / Photoshop / Affinity / GIMP / Krita 로 편집. 저장하면 자동으로 다시 불러오기
- **미리보기**：카메라 워크＋페이드가 적용된 비디오 콘티 재생. Dialogue 패널 제공
- **콘티 인쇄(Print Storyboard)**：애니메이션 스튜디오 스타일 레이아웃으로 A4 인쇄·PDF 출력
- **동영상 내보내기**：WebCodecs 로 네이티브 해상도·fps 의 MP4(H.264) 내보내기
- **Undo / Redo**：편집 작업의 취소·다시 실행
- **다국어 지원(i18n)**：日本語 / 한국어 / English
- **라이트·다크 테마**：취향에 맞게 전환

## 동작 환경

데스크톱 버전(Electron / Mac·Windows·Linux)과 설치가 필요 없는 Web 버전을 모두 제공합니다.
