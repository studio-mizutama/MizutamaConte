# 기본 사용법

## 새로 만들기·열기

- **새로 만들기**：프로젝트 이름·저장 폴더·해상도·화면비(SD / HD / FHD / 2K / 4K × 4:3 / 16:9 / 1.85:1 / 2.39:1)를 선택하여 생성합니다.
- **열기**：기존 프로젝트 폴더를 선택하여 엽니다.
- 데이터는 **PSD ＋ JSON** 으로 자동 저장됩니다.

## 편집(Edit)

- **행 추가·씬 구분**：콘티에 행이나 구분선을 추가합니다.
- **레이어**：하나의 CUT 안에 여러 레이어(시계열 플립북)를 둘 수 있습니다.
- **🔗 결합·➖ 분리**：인접한 CUT 을 결합하거나 분리할 수 있습니다.
- **인라인 편집**：ACTION / DIALOGUE / TIME 을 그 자리에서 직접 편집할 수 있습니다.

## 트랜지션과 카메라 워크

- **트랜지션**：페이드인／페이드아웃에 더해 크로스를 설정할 수 있습니다. **크로스는 인접한 2 개의 CUT 에서 공유**됩니다.
- **카메라 워크**：CUT 마다 팬·줌을 설정합니다. 슬라이더와 수치 입력으로 조정할 수 있습니다.

## 캔버스 리사이즈·크롭

캔버스를 리사이즈하면, 변경한 방향에 맞는 **기본 카메라 워크가 자동 생성** 됩니다. 크롭으로 구도를 잘라낼 수도 있습니다.

## 외부 드로잉 앱으로 그리기

- **PICTURE 를 더블클릭** 하면 CLIP STUDIO PAINT / Photoshop / Affinity / GIMP / Krita 등으로 열어 편집할 수 있습니다.
- 외부 앱에서 저장하면 Mizutama Conte 가 **자동으로 다시 불러옵니다**.

## 미리보기

- **카메라 워크＋페이드가 적용된** 비디오 콘티를 재생합니다.
- **Dialogue 패널** 로 대사를 확인할 수 있습니다.

## 콘티 인쇄·동영상 내보내기

- **콘티 인쇄(Print Storyboard)**：애니메이션 스튜디오 스타일 레이아웃으로 A4 인쇄·PDF 출력합니다.
- **동영상 내보내기**：네이티브 해상도·fps 로 MP4(H.264) 를 내보냅니다.
- 둘 다 Web 버전에서는 **Share 메뉴**, Electron 버전에서는 **File 메뉴** 에서 실행할 수 있습니다. 단축키는 **⌘P**(인쇄)／ **⌘E**(내보내기) 입니다.

## Undo / Redo

- 단축키 **⌘Z**(취소)／ **⌘⇧Z**(다시 실행).
- **Edit 메뉴**, 또는 툴바의 버튼에서도 실행할 수 있습니다.
