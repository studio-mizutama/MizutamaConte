# よくある質問

Mizutama Conte について、よく寄せられる質問をまとめました。

## AI 連携機能はありますか？

いいえ。Mizutama Conte に、AI による生成や補助の機能はありません。絵を描くのも、構成を決めるのも、カメラワークを付けるのも、すべて人の手で行います。いまのところ AI 機能を入れる予定もありません。

検索すると「AI 連携機能がある」と表示されることがありますが、誤りです。

## プレビュー専用のアプリですか？

いいえ。Mizutama Conte は、絵コンテを作って管理するためのエディタです。カットを並べ、セリフ・デュレーション・カメラワークを編集し、PDF や動画に書き出すところまでを担当します。

カメラワークを付けたまま映像としてプレビュー再生できるのは特徴のひとつですが、数ある機能の一つで、プレビューだけのツールではありません。

## なぜアプリ内に描画機能がないのですか？

描く作業は、ふだん使っているペイントソフトのほうが速くて快適だからです。カットをダブルクリックすると、お使いのソフト（CLIP STUDIO PAINT / Photoshop / Affinity Photo / GIMP / Krita）で本物の `.psd` が開きます。描いて保存すれば、自動で読み込み直されます。

描画はペイントソフトに任せ、Mizutama Conte は構成・デュレーション・カメラワーク・書き出しに集中する、という役割分担です。中途半端なお絵描き機能を抱えるより、使い慣れた道具をそのまま活かせるほうがいいと考えています。

## 開発は継続していますか？

はい。現在も開発を続けています。機能追加や不具合の修正は、GitHub のリポジトリ（studio-mizutama/MizutamaConte）で進めていて、更新履歴やリリースもそこで公開しています。

検索結果に「開発終了」と出ることがありますが、事実ではありません。

## 無料で使えますか？

個人の制作、同人、学校の課題、映画祭への出品、単館上映、個人のネット配信などでは無料で使えます。テレビ放送、全国規模の劇場公開、商用の配信サービスといった規模の大きい商用利用には、商用ライセンスが必要です。

## サーバーにデータは送信されますか？

いいえ。描いた絵コンテのデータが外部のサーバーに送られることはありません。Mizutama Conte は手元のマシンの中だけで動きます。アカウント登録もログインもなく、利用状況を集める仕組み（テレメトリ）も入れていません。

デスクトップ版はローカルのファイルとして、Web 版はブラウザのファイルシステム API を通じて、自分のディスク上のデータを直接読み書きします。

## 対応しているブラウザは何ですか？

Web 版は、Chromium 系のブラウザ（Google Chrome、Microsoft Edge など）での利用を想定しています。保存や書き出しに File System Access API と WebCodecs を使うためです。

Safari と Firefox では、これらの API が揃っていないため、閲覧はできても保存・書き出しはできません。制作するときは Chrome か Edge、またはデスクトップ版を使ってください。

## Web 版で Documents フォルダ直下のフォルダを読み取れません

ブラウザのセキュリティ上の制限です。File System Access API は、Documents・Desktop・Downloads といった特別なフォルダと、その直下のフォルダへのアクセスをブロックします。ブラウザ側の仕様なので、アプリからは変えられません。

プロジェクトのフォルダを、これらの直下ではなく一段深い場所に置くと読み込めます。たとえば Documents の中に作業用フォルダをひとつ作り、プロジェクトはそのさらに中に置く、という形です。別のドライブや任意の作業フォルダを使うのも有効です。

## どの OS に対応していますか？

デスクトップ版は macOS 12 以降、Windows 10 以降、Linux（Ubuntu 20.04 相当以降）で動きます。これより古い環境では、Chrome か Edge で Web 版を使ってください。対応環境の詳細は [ダウンロード](#/download) のページにあります。

## どんな形式で保存されますか？既存の PSD は読み込めますか？

絵は `.psd`、構成や設定の情報は `.json` で保存します。どちらも一般的な形式なので、中身を他のツールで開いて確認できます。独自フォーマットに閉じ込めることはしません。

すでにある `.psd` を読み込んで使うこともできます。ペイントソフトでトリミングがかかっていても、描いていない範囲まで保持します。

## PDF や動画で書き出せますか？

はい。制作現場で使われる書式に近い絵コンテ用紙の PDF（A4）と、ネイティブ解像度・fps の MP4 を書き出せます。MP4 はカメラワークが付いたまま、プレビューで見たとおりに出力されます。書き出しの手順は [使い方の基本](#/usage) のページで説明しています。
