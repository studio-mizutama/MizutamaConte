> ダウンロードは **必ず公式リポジトリ（studio-mizutama/MizutamaConte）からのみ** 行ってください。第三者が再配布したバイナリは使用しないでください。

## macOS の初回起動手順

未署名アプリは Gatekeeper によって最初の起動がブロックされます。以下のいずれかの方法で起動してください。

### 方法 1：Finder から開く（推奨）

1. `Mizutama Conte.app` を **右クリック（Control + クリック）** します。
2. メニューから **「開く」** を選びます。
3. 表示されたダイアログで再度 **「開く」** を選びます。

一度この手順で起動すれば、次回以降は通常どおりダブルクリックで開けます。

### 方法 2：ターミナルで quarantine 属性を削除

アプリを `アプリケーション` フォルダに移動したうえで、ターミナルで次のコマンドを実行します。

```sh
xattr -dr com.apple.quarantine "/Applications/Mizutama Conte.app"
```

### その他の選択肢

- リポジトリをクローンして **自分でビルド** する。
- インストール不要の **Web 版**（後述）を使う。

## Windows の初回起動手順

未署名のため、SmartScreen が「Windows によって PC が保護されました」と表示することがあります。

1. ダイアログの **「詳細情報」** をクリックします。
2. **「実行」** をクリックします。
