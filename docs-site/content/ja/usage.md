# 使い方の基本

## 新規作成・開く

- **新規作成**：プロジェクト名・保存先フォルダ・解像度・アスペクト比（SD / HD / FHD / 2K / 4K × 4:3 / 16:9 / 1.85:1 / 2.39:1）を選んで作成します。
- **開く**：既存のプロジェクトフォルダを選んで開きます。
- データは **PSD ＋ JSON** で自動保存されます。

## 編集（Edit）

- **行の追加・シーン区切り**：絵コンテに行や区切りを追加します。
- **レイヤー**：1 つの CUT の中に複数のレイヤー（時系列のフリップブック）を持たせられます。
- **🔗 結合・➖ 分離**：隣り合う CUT を結合したり、分離したりできます。
- **インライン編集**：ACTION / DIALOGUE / TIME をその場で直接編集できます。

## トランジションとカメラワーク

- **トランジション**：フェードイン／フェードアウトに加え、クロスを設定できます。**クロスは隣り合う 2 つの CUT で共有**されます。
- **カメラワーク**：CUT ごとにパン・ズームを設定します。スライダーと数値入力で調整できます。

## キャンバスのリサイズ・クロップ

キャンバスをリサイズすると、変更した方向に応じた **デフォルトのカメラワークが自動生成** されます。クロップで構図を切り出すこともできます。

## 外部お絵描きアプリで描く

- **PICTURE をダブルクリック** すると、CLIP STUDIO PAINT / Photoshop / Affinity / GIMP / Krita などで開いて編集できます。
- 外部アプリで保存すると、Mizutama Conte 側が **自動で再読み込み** します。

## プレビュー

- **カメラワーク＋フェード付き** でビデオコンテを再生します。
- **Dialogue パネル** でセリフを確認できます。

## 絵コンテ印刷・動画書き出し

- **絵コンテ印刷（Print Storyboard）**：アニメスタジオ風レイアウトで A4 印刷・PDF 出力します。
- **動画書き出し**：ネイティブ解像度・fps で MP4（H.264）を書き出します。
- いずれも、Web 版では **Share メニュー**、Electron 版では **File メニュー** から実行できます。ショートカットは **⌘P**（印刷）／ **⌘E**（書き出し）です。

## Undo / Redo

- ショートカット **⌘Z**（取り消し）／ **⌘⇧Z**（やり直し）。
- **Edit メニュー**、またはツールバーのボタンからも実行できます。
