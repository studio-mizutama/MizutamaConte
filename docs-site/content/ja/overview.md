# はじめに

**Mizutama Conte** は、絵コンテの作成・管理を行うデスクトップ／Web アプリです。

最大の特長は、**カメラワーク（パン・ズーム）付きのビデオコンテ（アニマティック）をプレビューできる**こと。レイアウトを並べるだけでなく、実際に動く映像として絵コンテを確認できます。これは類似ツールにはあまりない機能です。

## 主な機能

- **解像度・アスペクト比**：SD / HD / FHD / 2K / 4K と 4:3 / 16:9 / 1.85:1 / 2.39:1 を組み合わせて新規プロジェクトを作成
- **編集（Edit）**：行の追加、シーン区切り、CUT 内の複数レイヤー（時系列フリップブック）、🔗 CUT の結合・分離、ACTION / DIALOGUE / TIME のインライン編集
- **カメラワーク**：CUT ごとにパン・ズームを設定。スライダーと数値入力で調整
- **トランジション**：フェードイン／フェードアウト、隣り合う CUT 間で共有するクロス
- **キャンバスのクロップ・リサイズ**：リサイズ時に方向に応じたデフォルトのカメラワークを自動生成
- **外部お絵描きアプリ連携**：PICTURE をダブルクリックして CLIP STUDIO PAINT / Photoshop / Affinity / GIMP / Krita で編集。保存すると自動で再読み込み
- **プレビュー**：カメラワーク＋フェード付きでビデオコンテを再生。Dialogue パネル付き
- **絵コンテ印刷（Print Storyboard）**：アニメスタジオ風レイアウトで A4 印刷・PDF 出力
- **動画書き出し**：WebCodecs によりネイティブ解像度・fps で MP4（H.264）を書き出し
- **Undo / Redo**：編集操作をやり直し・取り消し
- **多言語対応（i18n）**：日本語 / 한국어 / English
- **ライト・ダークテーマ**：好みに合わせて切り替え

## 動作環境

デスクトップ版（Electron / Mac・Windows・Linux）と、インストール不要の Web 版の両方を用意しています。
