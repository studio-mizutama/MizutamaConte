# はじめに

Mizutama Conte は、アニメ制作の絵コンテを作るためのエディタです。Mac / Windows / Linux で動く Electron 版と、ブラウザで動く Web 版があります。

絵コンテをただ並べて印刷するだけでなく、カメラワークを付けたまま映像として再生できるのが特徴です。データはすべて手元のマシンで完結し、サーバーに送りません。無料で使えます。

## このアプリでできること

カットの絵を描き、セリフやデュレーション、カメラワークを付けて、1 本の絵コンテにまとめます。仕上げた絵コンテは、制作現場で標準的に使われる書式に近い PDF や、実際に動く MP4 として書き出せます。

絵を描く部分は外部のペイントソフトに任せ、Mizutama Conte は構成・デュレーション・カメラワーク・書き出しを担当します。

## 3 つの考え方

### カメラワーク付きで再生できる

パンやズームを設定すると、その動きが実際の映像としてプレビュー再生されます。静止画の絵コンテだけでは伝わりにくい「カメラの動き」を、作っている途中で確認できます。プレビューと書き出した MP4 の見えかたは一致します。

### 手元で完結する

ファイルはローカルに保存され、外部に送信されません。アカウント登録もありません。Electron 版はローカルのファイルとして、Web 版はブラウザのファイルシステム API を通じて、自分のディスク上のデータを直接扱います。

### 自分の使っているペイントソフトで描く

カットをダブルクリックすると、ふだん使っているペイントソフト（CLIP STUDIO PAINT / Photoshop / Affinity Photo / GIMP / Krita）で本物の `.psd` が開きます。描いて保存すれば自動で読み込み直されます。独自フォーマットに変換することはありません。ペイントソフト側でトリミングがかかっても、描いていない範囲まで保持します。

## 主要な機能

- 解像度とアスペクト比の選択：SD / HD / FHD / 2K / 4K と、4:3 / 16:9 / 1.85:1 / 2.39:1 の組み合わせを作成時に選びます。
- クロップとリサイズ：Shift で縦・横・比率にスナップします。リサイズした向きから、パンやズームの既定カメラワークを自動で作ります。
- カメラワーク編集：スライダーと数値入力で調整でき、IN / OUT の入れ替えもできます。ネイティブ解像度の値はグレーアウトします。
- ストップウォッチ Rec：左ツールバーの T ツールです。行ごとに Rec を出し、Space で計測の開始・停止をします。芝居を演じた長さが、そのままカットのデュレーション（秒 + コマ）になります。
- レイヤー：カット内の時系列フリップブックとして扱います。背景の上に重ねたレイヤーが 1 コマずつ送られます。
- カット操作：行の追加、シーン区切り、カットの結合・分離、レイヤーの追加、ACTION / DIALOGUE / TIME の編集ができます。
- トランジション：フェード（White / Black の In / Out）と、クロスフェードに対応します。
- 書き出し：スタジオ書式の絵コンテ用紙 PDF（A4）と、ネイティブ解像度・fps の MP4 を出力できます。
- 自動保存（PSD + JSON）、新規作成、最近開いたプロジェクト、Undo / Redo、多言語（日本語 / 한국어 / English）、ライト・ダークテーマに対応します。

## こんな人向け

- カメラの動きまで含めて絵コンテを確認したい人
- データを自分の手元に置いておきたい人
- ふだん使っているペイントソフトを変えずに絵コンテを作りたい人
- 後工程に渡せる書式の PDF や、確認用の動画が必要な人

## 対応環境

Electron 版は macOS 12 以降 / Windows 10 以降 / Linux（Ubuntu 20.04 相当以降）で動きます。これより古い環境では、モダンな Chrome / Edge で Web 版を使ってください。

配布しているビルドは未署名のため、初回起動には一手間かかります。手順はダウンロードのページで説明します。
