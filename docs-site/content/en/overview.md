# Overview

**Mizutama Conte** is a desktop and web app for creating and managing storyboards (絵コンテ).

Its standout feature is the ability to **preview a video conte (animatic) with camera work — pan and zoom — applied**. Instead of just laying out frames, you can watch your storyboard as actual moving footage. This is rare among comparable tools.

## Main Features

- **Resolution & aspect ratio**: create new projects by combining SD / HD / FHD / 2K / 4K with 4:3 / 16:9 / 1.85:1 / 2.39:1
- **Editing (Edit)**: add rows, insert scene dividers, multiple layers per CUT (a time-series flipbook), 🔗 merge / split CUTs, and inline ACTION / DIALOGUE / TIME editing
- **Camera work**: set pan and zoom per CUT, adjusted with sliders and numeric input
- **Transitions**: fade in / fade out, plus a cross that is shared between adjacent CUTs
- **Canvas crop & resize**: resizing auto-generates default camera work based on direction
- **External paint app integration**: double-click a PICTURE to edit it in CLIP STUDIO PAINT / Photoshop / Affinity / GIMP / Krita; it auto-reloads when you save
- **Preview**: play the video conte with camera work and fades, alongside a Dialogue panel
- **Print Storyboard (絵コンテ印刷)**: A4 print and PDF output in an anime-studio storyboard layout
- **Export Video**: MP4 (H.264) at native resolution and fps via WebCodecs
- **Undo / Redo**: undo and redo your edits
- **Internationalization (i18n)**: 日本語 / 한국어 / English
- **Light & dark theme**: switch to suit your preference

## Availability

Available as a desktop build (Electron for Mac, Windows, and Linux) and as a web version that needs no installation.
