# Basic Usage

## New / Open

- **New**: create a project by choosing a name, a destination folder, a resolution, and an aspect ratio (SD / HD / FHD / 2K / 4K × 4:3 / 16:9 / 1.85:1 / 2.39:1).
- **Open**: select an existing project folder to open it.
- Your data is auto-saved as **PSD + JSON**.

## Editing (Edit)

- **Add rows & scene dividers**: add rows or dividers to the storyboard.
- **Layers**: a single CUT can hold multiple layers (a time-series flipbook).
- **🔗 Merge / ➖ Split**: merge adjacent CUTs together, or split them apart.
- **Inline editing**: edit ACTION / DIALOGUE / TIME directly in place.

## Transitions & Camera Work

- **Transitions**: set fade in / fade out, plus cross. **A cross is shared between the two adjacent CUTs.**
- **Camera work**: set pan and zoom per CUT, adjusted with sliders and numeric input.

## Canvas Resize / Crop

Resizing the canvas **auto-generates default camera work** based on the direction you changed. You can also crop to reframe the composition.

## Drawing in an External Paint App

- **Double-click a PICTURE** to open it in CLIP STUDIO PAINT / Photoshop / Affinity / GIMP / Krita and edit it there.
- When you save in the external app, Mizutama Conte **auto-reloads** the image.

## Preview

- Play the video conte **with camera work and fades**.
- Check your lines in the **Dialogue panel**.

## Print Storyboard / Export Video

- **Print Storyboard (絵コンテ印刷)**: A4 print and PDF output in an anime-studio storyboard layout.
- **Export Video**: export MP4 (H.264) at native resolution and fps.
- Run both from the **Share menu** on the web version, or the **File menu** in the Electron app. Shortcuts are **⌘P** (print) and **⌘E** (export).

## Undo / Redo

- Shortcuts: **⌘Z** (undo) / **⌘⇧Z** (redo).
- Also available from the **Edit menu** or the toolbar buttons.
