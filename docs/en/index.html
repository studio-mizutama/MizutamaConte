Runs locally

Every file stays on your own machine. No account, no login.

Draw in paint software you already know

CLIP STUDIO PAINT, Photoshop, Affinity Photo, GIMP, Krita. Keep drawing in the tools you already use.

Native PSD editing

Double-click a cut and the actual .psd opens in your paint software. Save it, and Mizutama Conte reloads it automatically and updates the preview. No proprietary format at all.

  • Downstream stages — layout, key animation, backgrounds — can use it straight away as a rough
  • Files sit plainly in a folder

Real resolutions and aspect ratios

Supports SD / HD / FHD / 2K / 4K and 4:3 / 16:9 / 1.85:1 / 2.39:1. Widen or stretch the canvas and camera work (pans and zooms) is inserted automatically.

  • Shift-drag to snap to vertical, horizontal, or ratio
  • A default camera move is generated from the resize direction

Stopwatch input

Enter durations efficiently with a stopwatch.

  • Start and stop each row with Space
  • Manual entry works too

A storyboard skeleton from your script

Import a script written in Markdown and get from script to storyboard faster.

  • Headings split scenes, horizontal rules split cuts, automatically
  • Dialogue and action are read in separately

On paper, in PDF, or as video

Print the storyboard sheet or export it to PDF. Render video at native resolution.

  • Print A4 studio-style storyboard sheets
  • Export video at native resolution and fps
