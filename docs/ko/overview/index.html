시작하기

Mizutama Conte는 애니메이션 제작용 콘티를 만들기 위한 에디터입니다. Mac / Windows / Linux에서 동작하는 Electron 버전과, 브라우저에서 동작하는 Web 버전이 있습니다.

콘티를 그저 나열해 인쇄하는 데 그치지 않고, 카메라 워크를 적용한 채로 영상으로 재생할 수 있는 것이 특징입니다. 데이터는 모두 사용자의 기기 안에서 처리되며 서버로 전송하지 않습니다. 무료로 사용할 수 있습니다.

이 앱으로 할 수 있는 것

컷의 그림을 그리고, 대사와 듀레이션, 카메라 워크를 더해 한 편의 콘티로 정리합니다. 완성한 콘티는 제작 현장에서 표준적으로 쓰이는 서식에 가까운 PDF나, 실제로 움직이는 MP4로 내보낼 수 있습니다.

그림을 그리는 부분은 외부 페인트 앱에 맡기고, Mizutama Conte는 구성, 듀레이션, 카메라 워크, 내보내기를 담당합니다.

세 가지 관점

카메라 워크를 적용해 재생할 수 있다

팬이나 줌을 설정하면 그 움직임이 실제 영상으로 미리보기 재생됩니다. 정지 화면의 콘티만으로는 전달하기 어려운 "카메라의 움직임"을 작업 도중에 확인할 수 있습니다. 미리보기와 내보낸 MP4의 보이는 방식은 일치합니다.

자신의 기기 안에서 완결된다

파일은 로컬에 저장되며 외부로 전송되지 않습니다. 계정 등록도 없습니다. Electron 버전은 로컬 파일로, Web 버전은 브라우저의 File System Access API를 통해, 자신의 디스크에 있는 데이터를 직접 다룹니다.

자신이 쓰는 페인트 앱으로 그린다

컷을 더블 클릭하면 평소에 쓰는 페인트 앱(CLIP STUDIO PAINT / Photoshop / Affinity Photo / GIMP / Krita)로 진짜 .psd가 열립니다. 그려서 저장하면 자동으로 다시 읽어들입니다. 독자 포맷으로 변환하지 않습니다. 페인트 앱 쪽에서 트리밍이 적용되어도, 그리지 않은 범위까지 그대로 보존합니다.

주요 기능

  • 해상도와 화면비 선택: SD / HD / FHD / 2K / 4K와, 4:3 / 16:9 / 1.85:1 / 2.39:1의 조합을 생성 시 선택합니다.
  • 크롭과 리사이즈: Shift로 세로, 가로, 비율에 스냅됩니다. 리사이즈한 방향에 맞춰 팬이나 줌의 기본 카메라 워크를 자동으로 만듭니다.
  • 카메라 워크 편집: 슬라이더와 수치 입력으로 조정할 수 있으며, IN / OUT을 서로 바꿀 수도 있습니다. 네이티브 해상도 값은 비활성화(회색)됩니다.
  • 스톱워치 Rec: 왼쪽 도구 막대의 T 도구입니다. 행마다 Rec을 표시하고, Space로 측정을 시작·정지합니다. 연기한 길이가 그대로 컷의 듀레이션(초 + 프레임)이 됩니다.
  • 레이어: 컷 안의 시계열 플립북으로 다룹니다. 배경 위에 겹친 레이어가 한 프레임씩 넘어갑니다.
  • 컷 조작: 행 추가, 씬 구분, 컷의 결합·분리, 레이어 추가, ACTION / DIALOGUE / TIME 편집을 할 수 있습니다.
  • 트랜지션: 페이드(White / Black의 In / Out)와 크로스페이드를 지원합니다.
  • 내보내기: 스튜디오 서식의 콘티 용지 PDF(A4)와, 네이티브 해상도·fps의 MP4를 출력할 수 있습니다.
  • 자동 저장(PSD + JSON), 새로 만들기, 최근 연 프로젝트, Undo / Redo, 다국어(한국어 / 일본어 / 영어), 라이트·다크 테마를 지원합니다.

이런 분에게

  • 카메라의 움직임까지 포함해 콘티를 확인하고 싶은 분
  • 데이터를 자신의 기기에 두고 싶은 분
  • 평소에 쓰는 페인트 앱을 바꾸지 않고 콘티를 만들고 싶은 분
  • 후공정(레이아웃·원화·배경)에 넘길 수 있는 서식의 PDF나, 확인용 동영상이 필요한 분

지원 환경

Electron 버전은 macOS 12 이상 / Windows 10 이상 / Linux(Ubuntu 20.04 상당 이상)에서 동작합니다. 이보다 오래된 환경에서는 최신 Chrome / Edge로 Web 버전을 사용하세요.

배포하는 빌드는 서명되지 않았기 때문에, 첫 실행에는 약간의 수고가 필요합니다. 절차는 다운로드 페이지에서 설명합니다.
