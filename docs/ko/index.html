콘티를 그리면 그대로 애니매틱(영상으로 재생되는 콘티)으로. 팬·줌이 들어간 콘티를 그 자리에서 재생하고, PDF나 영상으로.

카메라·편집 효과를 그 자리에서 확인.

카메라 워크가 들어간 프리뷰

타이밍과 카메라 움직임을 그 자리에서 재생해 확인할 수 있습니다.

로컬에서 완결

파일은 전부 사용자의 기기 안에. 계정도 로그인도 필요 없습니다.

쓰던 페인트 앱 그대로

CLIP STUDIO PAINT·Photoshop·Affinity Photo·GIMP·Krita. 그림은 손에 익은 도구로 그대로.

네이티브 PSD 편집

컷을 더블클릭하면 실제 .psd가 쓰던 페인트 앱에서 열립니다. 저장하면 자동으로 다시 읽어들여 프리뷰에 반영. 독자 포맷은 일절 없습니다.

  • 후공정(레이아웃·원화·배경미술)을 그대로 밑그림으로 쓸 수 있다
  • 파일은 폴더에 그대로 나열된다

실제 제작용 해상도·화면비

SD / HD / FHD / 2K / 4K, 4:3 / 16:9 / 1.85:1 / 2.39:1을 지원. 캔버스를 넓히거나 늘리면 카메라 워크(팬·줌)를 자동 삽입.

  • Shift 드래그로 세로·가로·비율에 스냅
  • 리사이즈 방향에 맞춰 기본 카메라를 자동 생성

스톱워치 입력

스톱워치로 효율적으로 듀레이션을 입력.

  • 행마다 Space로 시작·정지
  • 직접 입력도 가능

각본에서 콘티의 뼈대를

Markdown으로 쓴 각본 임포트를 지원. 각본에서 콘티 작성을 효율적으로.

  • 제목으로 신, 수평선으로 컷을 자동 분할
  • 대사·액션을 나눠서 읽어들임

종이로도, PDF로도, 영상으로도

콘티 용지는 인쇄·PDF 내보내기가 가능. 영상은 네이티브 해상도로.

  • A4 스튜디오 서식의 콘티 인쇄
  • 네이티브 해상도·fps의 영상 내보내기
