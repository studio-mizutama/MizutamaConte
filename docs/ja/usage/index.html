使い方の基本

Mizutama Conte は、アニメ制作向けに、絵コンテを描いてビデオコンテとして再生できるエディタです。データはすべて手元のパソコンの中だけで処理され、サーバーには送られません。デスクトップ版（Mac / Windows / Linux）と、ブラウザで動作する Web 版を提供しています。

このページは、プロジェクトを作ってから書き出すまでの一通りの流れを順番に説明します。キーボードショートカットの一覧は別ページ（ショートカット）にまとめてあります。

全体の流れ

  1. プロジェクトを作る（解像度・アスペクト比を選ぶ）
  2. カットの絵を描く（外部のペイントソフトで PSD ファイルを編集）
  3. カットを編集する（カットの追加・シーン区切りの編集・カットの結合と分離・レイヤーの追加、ACTION / DIALOGUE / TIME の編集）
  4. カメラワークを付ける（クロップとリサイズ、スライダー編集）
  5. デュレーションを決める（ストップウォッチで実演して計測）
  6. トランジションを付ける（フェード・クロスフェード）
  7. プレビューで再生して確認する
  8. PDF や MP4 に書き出す

途中の保存は自動で行われます（PSD と JSON）。明示的な保存操作は基本的に不要です。

1. プロジェクトを作る

新規プロジェクト

メニューから「新規プロジェクト」を選びます。作成時に、これからの絵コンテの解像度とアスペクト比を決めます。

解像度は SD / HD / FHD / 2K / 4K から、アスペクト比は 4:3 / 16:9 / 1.85:1 / 2.39:1 から選びます。ここで決めた解像度が PSD の基準サイズ（ネイティブ解像度）になります。

迷ったら、テレビアニメなら HD または FHD の 16:9、劇場作品を想定するなら 1.85:1 または 2.39:1 あたりが目安です。

脚本（Markdown）から作る

書いた脚本がすでに Markdown である場合は、そこから一気にシーンとカットを起こせます。メニューの「脚本から新規」（Cmd / Ctrl + Shift + N）から読み込みます。見出しがシーンの区切り、水平線がカットの区切り、セリフの行が DIALOGUE になり、各カットに空の PSD が自動で作られます。

記法の詳細と具体例は 脚本（Markdown）の書き方 を参照してください。

2. 絵を描く

Mizutama Conte 自体には描画ツールはありません。絵はふだん使っているペイントソフトで描きます。 CLIP STUDIO PAINT / Photoshop / Affinity Photo / GIMP / Krita など、PSD ファイルを読み込んで上書き保存できるものであれば、どんなソフトでも使えます。

PSD を開いて描く

  1. カットのコマ（PICTURE）をダブルクリックします。
  2. 設定で選んだペイントソフトが起動し、そのカットの実体である .psd ファイルが開きます。
  3. そのまま描いて、上書き保存します。
  4. Mizutama Conte に戻ると、保存された内容が自動で読み込まれ、コマの表示が更新されます。

独自フォーマットには変換していません。開かれるのは普通の PSD で、いつものソフトでそのまま編集できます。

3. カットを編集する

絵コンテはシーンとカット（CUT）の集まりです。一覧の各行がひとつのカットにあたります。

カットを追加する

行を追加すると、新しい空のカットが入ります。各カットには空の PSD が作られるので、そのまま絵を描き始められます。

シーンを区切る

任意の位置でシーンを区切れます。シーン区切りを入れると、そこから新しいシーンが始まります。

カットを結合・分離する

隣り合うカットは結合できます。逆に、結合したカットは分離して元に戻せます。割り直しのときに使います。

レイヤー＝カット内のコマ

ひとつのカットは複数のレイヤーを持てます。ここでのレイヤーは、カット内の時系列のフリップブック（パラパラ漫画）として扱われます。背景の上に重ねたレイヤーが、1 コマずつ順に表示されます。中割りや動きのあるカットを、1 カットの中で表現するための仕組みです。複数のレイヤーをまとめて 1 コマとして扱いたい場合は、レイヤーをグループ化します。

ACTION / DIALOGUE / TIME

各カットには次の情報を書き込めます。

  • ACTION（ト書き）: カットの内容や動きの説明
  • DIALOGUE（セリフ）: そのカットで話されるセリフ
  • TIME（デュレーション）: そのカットの長さ

ACTION と DIALOGUE はその場でテキストを編集できます。TIME は数値で入力するほか、後述のストップウォッチでも決められます。

4. カメラワークを付ける

カット上でカメラ枠（クロップ枠）をドラッグして大きさと位置を決めます。枠を大きくするとズームイン、位置をずらせばパンを表現できます。

ドラッグ中に Shift を押すと、スナップが効きます。

  • 横方向のスナップ（水平のパン）
  • 縦方向のスナップ（垂直のパン）
  • 比率を保ったスナップ（アスペクト比を保ったままのズーム）

リサイズの向きから既定のカメラワークを自動生成

カメラ枠をリサイズすると、その向き（どちらに広げた / 縮めたか）から、もっともらしいパンやズームの初期値が自動で入ります。まずざっくり枠を動かして当たりを付け、あとから細かく調整する、という流れが楽です。

スライダーと数値で調整する

カメラワークパネルでは、スライダーと数値入力の両方で値を詰められます。

  • スライダーで大まかに合わせ、数値で正確に決める
  • IN と OUT は入れ替え（スワップ）できる
  • ネイティブ解像度いっぱい（=切り取りなし、動きなし）の状態の項目はグレーアウトされる

グレーアウトは「ここはカメラワークがかかっていない」ことの目印です。

5. デュレーションを決める（ストップウォッチ）

カットの長さは、実際に芝居を演じてみて、その時間をそのままデュレーションに入力できます。

  1. 左のツールバーで T（ストップウォッチ）ツールを選びます。
  2. デュレーションを計りたいカットを選びます。
  3. Space で計測を開始し、もう一度 Space で停止します。
  4. 演じていた長さが、そのカットの TIME（秒 + コマ）として入ります。

セリフを声に出して読む、動きを体で再現するなど、頭の中のタイミングをそのまま数値にできます。手で秒数を入力したい場合は、TIME を直接編集してもかまいません。

6. トランジションを付ける

カットの切り替わりに、フェードとクロスフェードを設定できます。

  • フェードイン・フェードアウト: White / Black の 2 種類を選べます。
  • クロスフェード: 前のカットと次のカットが重なって入れ替わるフェードです。

トランジションのデュレーションの既定値は、0.5 秒です（例: 24fps なら 12 コマ、30fps なら 15 コマ）。長さはあとから調整できます。

7. プレビューで再生する

プレビューを再生すると、設定したデュレーション・カメラワーク・トランジションがすべて反映された状態で、ビデオコンテとして通しで見ることができます。パンやズームは実際の動きとして再生されるので、タイミングや画面の運びを、絵を描いた段階で確認できます。

8. 書き出す

仕上がった絵コンテは、2 つの形式で書き出せます。

PDF 印刷（絵コンテ用紙）

スタジオの絵コンテ用紙に近い書式（A4）で書き出します。各カットの PICTURE には IN / OUT のカメラ枠が示され、累積デュレーションも入ります。紙の運用にそのまま乗せられます。

MP4 動画（ビデオコンテ）

プロジェクトのネイティブ解像度・fps で MP4 形式の動画ファイルとして書き出します。プレビューで確認した動きが、そのまま動画になります。

対応環境

デスクトップ版の対応環境は次のとおりです。

  • macOS 12 以降
  • Windows 10 以降
  • Linux（Ubuntu 20.04 相当以降）

これより古い環境では、Chromium 系ブラウザで動く Web 版を利用してください。

Web 版は Chromium 系ブラウザ（Chrome / Edge など）での利用を想定しています。ファイルの保存・読み込みや動画書き出しは File System Access API・WebCodecs に依存するため、Safari や Firefox では閲覧のみ（保存・書き出し不可）になります。PDF 印刷も Chromium 系を推奨します。

配布しているビルドは現状コード署名をしていないため、初回起動時に OS の警告を回避する一手間が必要です（Mac ならゲートキーパーの許可など）。

その他

  • 自動保存: 編集内容は PSD と JSON に随時保存されます。
  • 最近開いたプロジェクト: 前回までの作業をすぐ開き直せます。
  • Undo / Redo: 操作の取り消しとやり直しができます。
  • 表示言語: 日本語 / 韓国語 / 英語 を切り替えられます。
  • テーマ: ライト / ダークを切り替えられます。

操作のショートカットは ショートカット のページにまとめています。
