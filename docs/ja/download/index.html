ダウンロード

Mizutama Conte には、Mac / Windows / Linux で動くデスクトップ版と、ブラウザだけで動く Web 版があります。どちらもローカルで完結し、描いた絵コンテのデータをサーバーに送ることはありません。

最新版は GitHub Releases ページ から入手できます。各リリースに Mac / Windows / Linux 向けのバイナリを追加していきます。ダウンロードは公式リポジトリ（studio-mizutama/MizutamaConte）からのみ行ってください。第三者が再配布したバイナリは使わないでください。

迷ったら、まず Web 版をブラウザで開いて試すのがおすすめです。インストール不要で、すぐに触れます。普段使いするなら、自動保存や外部のペイントソフト連携が使えるデスクトップ版をどうぞ。

どれを使うか

  • まず試したい / インストールしたくない → Web 版（Chromium 系ブラウザを推奨）
  • 普段使いする / PSD を外部のペイントソフトで往復編集したい → デスクトップ版（Mac / Windows / Linux）

デスクトップ版のダウンロード

Releases ページ から、お使いの OS のファイルを選んでください。

macOS

  • .dmg をダウンロードします。
  • .dmg を開いて、Mizutama Conte を Applications フォルダにドラッグします。

Apple Silicon（M1 以降）と Intel の両方に対応しています。ファイル名にチップの区別がある場合は、お使いの Mac に合うものを選んでください。初回起動には下記「初回起動について」の手順が必要です。

Windows

  • インストーラ（.exe）をダウンロードします。
  • ダウンロードした .exe を実行し、画面の指示に従ってインストールします。
  • 初回実行時に SmartScreen の警告が出ます。下記「初回起動について」を参照してください。

Linux

  • AppImage をダウンロードします。
  • ファイルに実行権限を付けてから起動します。ファイルを右クリック →「プロパティ」→「実行可能にする」にチェックを入れるか、ターミナルで次を実行します。
chmod +x Mizutama-Conte*.AppImage

対応環境

  • macOS 12（Monterey）以降
  • Windows 10 以降
  • Linux: Ubuntu 20.04 相当以降

これより古い環境では、デスクトップ版が起動しないことがあります。その場合は Web 版を使ってください。

初回起動について（未署名ビルド）

現在配布しているデスクトップ版には、コード署名・公証（notarization）を付けていません。そのため初回起動時に、OS が「開発元を確認できない」などの警告を出します。アプリが壊れているわけではなく、署名されていないアプリ全般に出るものです。インストールと実行は自己責任でお願いします。次の手順で起動できます。

macOS

ダブルクリックすると「"Mizutama Conte" は壊れているため開けません」や「開発元を確認できないため開けません」と表示されることがあります。次のどちらかで起動できます。

方法 1: システム設定から許可する（推奨）

  1. Mizutama Conte をダブルクリックして、警告が出るのを確認します。ここではゴミ箱に入れず、いったん閉じます。
  2. アップルメニュー →「システム設定」→「プライバシーとセキュリティ」を開きます。
  3. 下の方に「"Mizutama Conte" は開発元を確認できないため、使用がブロックされました」といったメッセージが出ています。その横の「このまま開く」をクリックします。
  4. もう一度確認ダイアログが出たら「開く」を選びます。次回からは普通にダブルクリックで起動できます。

方法 2: ターミナルで quarantine 属性を外す

「壊れているため開けません」が出て方法 1 の表示が見当たらない場合は、ダウンロード時に付く隔離属性を外すと起動できます。アプリを Applications フォルダに移したうえで、ターミナルで次を実行してください。

xattr -dr com.apple.quarantine "/Applications/Mizutama Conte.app"

実行後、もう一度アプリを開きます。

Windows

初めて .exe を実行すると、Microsoft Defender SmartScreen が「Windows によって PC が保護されました」という青い画面を出すことがあります。

  1. 画面内の「詳細情報」をクリックします。
  2. 下に表示される「実行」ボタンを押します。

これでインストール（または起動）が進みます。次回以降は表示されません。

Linux

ディストリビューションによっては、起動時に実行権限や信頼の確認を求められることがあります。前述のとおり AppImage に実行権限を付けてから起動してください。

ファイルマネージャで実行権限を付けても、ダブルクリックではセキュリティの確認で起動できないことがあります。その場合は、ターミナルから直接実行すると起動できます。

./Mizutama-Conte*.AppImage

Web 版

ブラウザで開くだけで使えます。インストールは不要です。

  • https://studio-mizutama.github.io/MizutamaConte/

ファイルの読み書きにはブラウザの File System Access API を使います。モダンな Chromium 系ブラウザでの利用を想定しています。古い OS でデスクトップ版が動かないときの選択肢としても使えます。
