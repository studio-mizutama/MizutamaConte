## v0.9.1

v0.9.0 のバグ修正リリースです。 / v0.9.0 버그 수정 릴리스입니다. / Bug-fix release for v0.9.0.

### 日本語

- **macOS**: パッケージ版でバージョン管理（git / git-lfs）が「未インストール」と誤検出され、有効化できなかった問題を修正しました。
- **Windows / Linux**: メニューバーのハンバーガーボタン（☰）の左右の余白が不揃いだったのを修正しました。

### 한국어

- **macOS**: 패키지 버전에서 버전 관리(git / git-lfs)가 "설치되지 않음"으로 잘못 감지되어 활성화할 수 없던 문제를 수정했습니다.
- **Windows / Linux**: 메뉴 바의 햄버거 버튼(☰) 좌우 여백이 맞지 않던 문제를 수정했습니다.

### English

- **macOS**: Fixed version control (git / git-lfs) being wrongly detected as "not installed" in the packaged app, so it could not be enabled.
- **Windows / Linux**: Fixed uneven left/right spacing of the hamburger button (☰) in the menu bar.
